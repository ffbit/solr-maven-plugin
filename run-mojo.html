solr:run

Full name:

com.ffbit.maven.plugins:solr-maven-plugin:0.0.6:run

Description:

Run a new Apache Solr instance and executes it indefinably.

Attributes:

  • Requires a Maven project to be executed.

Optional Parameters

Name Type Since Description
contextPath String 0.0.1 The context path for the server instance.
Default value is: /solr.
jettyXml String 0.0.6 Comma separated list of a jetty xml configuration files whose contents will be applied before any plugin configuration. Optional.
loggingPropertiesPath File 0.0.6 Jetty login properties.
port int 0.0.1 The running port.
Default value is: 8983.
repositorySession RepositorySystemSession 0.0.1 The current repository/network configuration of Maven.
Default value is: ${repositorySystemSession}.
solrHome File 0.0.1 The Apache Solr Home directory.
Default value is: ${project.build.directory}/solr.
User property is: solrHome.
solrVersion String 0.0.1 The running Apache Solr version.
Default value is: 4.3.0.
